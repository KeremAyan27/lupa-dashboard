# Atlas — CEO Mobile Dashboard (Decision Support System)

Boğaziçi University MIS capstone project. A mobile-first PWA that consolidates
sales, stock, order and collection data behind four tabs, with a transparent
rule-based alert engine at its core.

Live Link: https://atlas-dashboard-sage.vercel.app

Stack: Next.js (App Router) · strict TypeScript · Tailwind CSS · Recharts · PWA · Vercel.

## Project structure

```
data/                 7 reproducible JSON files (read-only, see below)
scripts/              reference workbook + xlsx_to_json.py exporter
types/atlas.ts        TypeScript model matching the JSON schema exactly
lib/data.ts           server-side JSON loaders (module-level cache)
lib/alert-engine.ts   rule-based anomaly detection (R1/R2/R3 — no ML)
lib/use-api.ts        client fetch hook with loading/error states
app/api/*/route.ts    overview · sales · stock · payments · alerts
app/                  / · /sales · /stock · /payments · /alerts/[id] · /about
components/           AppShell (top bar, alert drawer, nav, toasts) + UI kit
```

## Data & reproducibility

The dataset (5,189 orders, 50 products, 200 customers, 4,850 payments, 1,616
stock movements, 01.01.2025–31.12.2025) is synthetic and reproducible: it was
generated by `generate_mock_data.py` with `random.seed(42)`, which also
exports the reference workbook. `scripts/xlsx_to_json.py` converts that
workbook (`scripts/CEO_Dashboard_Tam_VeriSeti.xlsx`) into `data/*.json` as a
pure format conversion — **never edit the JSON files by hand**:

```bash
pip install openpyxl && python3 scripts/xlsx_to_json.py
```

Intentional anomalies (March −39.5%, April +58.4%, October +23.7%, 5
critical-stock products, 12 overdue payments) are preserved end to end; the
alert engine derives every alert in the UI from this data at request time.
All "today"-relative logic is anchored to the dataset end (31.12.2025).

```bash
npx tsc --noEmit   # strict typecheck (no `any`)
npm run lint
npm run build
```

kept for reference only (excluded from linting, not imported anywhere).
